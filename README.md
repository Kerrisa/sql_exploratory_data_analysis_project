# sql_exploratory_data_analysis_project
Advanced SQL and EDA project built on a dimensional data warehouse model, showcasing fact-to-dimension joins, data validation, performance metrics, trend discovery, and insight-driven reporting using CTEs, subqueries, and window functions.


Core techniques include:
- Data profiling and completeness checks
- Fact-to-dimension relationship validation
- Advanced joins and subqueries
- Common Table Expressions (CTEs)
- Window functions
- KPI and trend analysis
- Change over time analysis
- Peformance analysis
- Data Segmentation
- Part-to whole analysis
- Magnitude and ranking analysis
- Dimension, Fact and database exploration 
- Outlier and anomaly detection
- Business intelligence reporting
